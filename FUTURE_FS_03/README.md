# 🍽️ FUTURE_FS_03 - Aangan Restaurant Website

🌐 **Live Demo:** https://future-intern-nxsl.vercel.app/

## 📌 Project Overview
This project is developed as **Task 3 - Local Business Website & Live Pitch Project** for the **Future Interns Full Stack Web Development Internship**.

The website is built for **Aangan Restaurant**, a local Indian cuisine restaurant located on **Ring Road, Ranchi, Jharkhand**.

It is designed to improve the restaurant’s:
- online presence
- menu visibility
- customer engagement
- local SEO reach
- mobile accessibility

---

## 🚀 Features
- ✅ Responsive modern restaurant landing page
- ✅ Attractive hero section with food branding
- ✅ About restaurant section
- ✅ Menu showcase section
- ✅ Contact / visit us section
- ✅ Smooth UI animations
- ✅ Mobile-friendly responsive design
- ✅ SEO-friendly structure
- ✅ Fast performance with Vite
- ✅ Reusable React components
