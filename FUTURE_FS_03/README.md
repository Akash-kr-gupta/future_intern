# FUTURE_FS_03 - Aangan Restaurant Website

## 📌 Project Overview
This project is developed as **Task 3 - Local Business Website & Live Pitch Project** for the **Future Interns Full Stack Web Development Internship**.

The website is built for **Aangan Restaurant**, a local Indian cuisine restaurant located on **Ring Road, Ranchi, Jharkhand**.

It is designed to help the business improve its **online presence, menu visibility, customer engagement, and local SEO reach**.

---

## 🚀 Features
- Responsive modern restaurant landing page
- Hero section with attractive food branding
- About section
- Menu showcase
- Contact / visit section
- Beautiful UI with animations
- Mobile-friendly design
- SEO-friendly structure
- Built with React + Vite

---

## 🛠️ Tech Stack
- React.js
- Vite
- CSS3
- JavaScript
- Responsive Web Design

---

## 🎯 Business Goal
This website helps the local business:
- Increase customer trust
- Improve local discoverability
- Showcase menu professionally
- Attract more dine-in customers
- Strengthen brand identity
